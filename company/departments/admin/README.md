# 管理部（総務）

## ミッション

会社の管理業務を効率的に運営し、全部門がスムーズに機能するための基盤を提供する。

## 責任範囲

- 総務業務全般
- 契約書・法務文書の管理
- ベンダー・外部委託先の管理
- 社内ツール・インフラの管理
- コンプライアンス管理
- 各種届出・手続き

## KPI

| 指標 | 目標 | 頻度 |
|------|------|------|
| 業務プロセス効率化率 | - | 四半期 |
| 契約管理精度 | 100% | 月次 |
| コンプライアンス遵守率 | 100% | 四半期 |

## 連携部門

- **財務部**: 経理業務、予算管理
- **人事部**: 労務管理
- **全部門**: 管理業務サポート
