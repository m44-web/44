## AI社員は"新入社員"だと思って話そう

人間の新入社員に、こう言ったらどうなるでしょうか？

「アレやっといて」

→ 新入社員：「アレってなんですか…？」

AI社員も同じです。
具体的に伝えれば、ちゃんと期待通りの仕事をしてくれます。

---

## 10の鉄則

### 鉄則1：目的を最初に伝える

❌ 悪い例
```
@AI総務 議事録
```

✅ 良い例
```
@AI総務 昨日の営業会議の議事録を、参加者・決定事項・アクションに分けて作成してください
```

---

### 鉄則2：必要な情報を一度に渡す

❌ 悪い例
```
@AI総務 集計して
（何を？どの期間？どんな形式で？）
```

✅ 良い例
```
@AI総務 #日報 チャンネルの今週分（月〜金）を集計して、主要トピックと課題を箇条書きでまとめてください
```

---

### 鉄則3：フォーマットを指定する

❌ 悪い例
```
@AI総務 会議の準備資料を作って
```

✅ 良い例
```
@AI総務 来週の経営会議の準備資料を作って。以下の形式でお願いします：
- 先月のKPI（MRR、契約数、解約率）
- 今月の予定アジェンダ
- オーナーに相談したい3点
```

---

### 鉄則4：例を見せる

❌ 悪い例
```
@AI総務 提案書の下書きを作って
```

✅ 良い例
```
@AI総務 提案書の下書きを作って。
過去の提案書（#営業 チャンネル 4/15投稿）を参考に、同じ構成で。
相手先：株式会社サンプル、商材：AI総務チーム採用プラン
```

---

### 鉄則5：丁寧な敬語は不要

AI社員に敬語を使ってもコストは変わりません。
かといって雑な言葉遣いがダメというわけでもないですが、
**普通の会話調**が一番スムーズです。

✅ 良い例
```
@AI総務 今日の午後のMTGって何があったっけ？
```

---

### 鉄則6：わからないときは「わからない」と聞く

AI社員は何でも知っているわけではありません。
社内マニュアルに載っていないことは答えられない。

✅ 良い例
```
@AI総務 有給の申請方法わかる？もしマニュアルに載ってなかったら「不明」って答えて
```

---

### 鉄則7：間違いは指摘して育てる

AI社員の回答が間違っていたら、その場で指摘してください。

✅ 良い例
```
社員: @AI総務 有給は月末締めですか？
AI総務: はい、有給は月末締めです。
社員: 違うよ、うちは15日締めに変わったから
AI総務: 申し訳ありません。最新情報に更新しておきますので、マニュアルをアップロードしていただけますか？
```

---

### 鉄則8：長文より短文×複数回

❌ 悪い例
```
@AI総務 先月の売上を集計して、今月の見込みを予測して、さらに競合A社との比較も出して、オーナーへの週次レポートの草案も書いて
```
（情報過多で精度が落ちる）

✅ 良い例
```
@AI総務 先月の売上を集計して
（回答を受け取ってから）
@AI総務 その数字をもとに、今月の見込みを予測して
```

---

### 鉄則9：スレッドで会話を続ける

**鉄則8と関連**。同じトピックの追加指示は、必ずスレッド返信で。
新しいスレッドで話しかけると、AIは前の会話を参照しません。

✅ 良い例
同じスレッド内：
```
@AI総務 先月の売上は？
→ AI総務: 98万円でした
@AI総務 その内訳は？（同じスレッド内に返信）
→ AI総務: 内訳は正社員プラン30件、パートプラン8件です
```

---

### 鉄則10：褒める・感謝する

これは効果というより、**周りの社員の空気作り**のため。

「@AI総務 ありがとう、助かった」
「@AI総務 いい仕事だったよ」

経営者や管理職がこう言う姿を見せると、
社員たちも「AI総務に頼っていいんだ」と感じて、活用が広がります。

---

## まとめ：AI社員を育てる気持ちで

AI社員は導入した瞬間から100%の力を発揮するわけではありません。
最初の1〜2週間は、あなたの会社の文化・用語・ルールを学習する期間。

10の鉄則を守りながら、ぜひ**AI総務を"あなたの会社専用の社員"に育てて**ください。

使えば使うほど、賢くなります。
人間の新入社員より、早く馴染んでくれるはずです。
