# 営業部

## ミッション

AIエージェント/SaaSプロダクトの顧客獲得と売上拡大を推進する。

## 責任範囲

- リード獲得・ナーチャリング
- 商談・クロージング
- 顧客ニーズのヒアリングと社内フィードバック
- 価格戦略の実行
- 売上目標の達成

## KPI

| 指標 | 目標 | 頻度 |
|------|------|------|
| MRR（月次定期収益） | - | 月次 |
| 新規顧客獲得数 | - | 月次 |
| 商談化率 | - | 月次 |
| 顧客単価（ARPU） | - | 月次 |
| 解約率（Churn Rate） | - | 月次 |

※目標値はOKR策定時に設定

## 連携部門

- **マーケティング部**: リード供給、コンテンツ連携
- **開発部**: 顧客フィードバックの共有、機能要望の伝達
- **財務部**: 売上報告、請求管理
