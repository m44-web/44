# 財務部

## ミッション

健全な財務管理と透明性のある数値管理で、経営判断を支える。

## 責任範囲

- 予算策定・管理
- 収支管理（売上・経費）
- 月次財務レポートの作成
- キャッシュフロー管理
- 財務予測・シミュレーション
- 税務対応

## KPI

| 指標 | 目標 | 頻度 |
|------|------|------|
| MRR（月次定期収益） | - | 月次 |
| バーンレート | - | 月次 |
| ランウェイ（資金余力） | - | 月次 |
| 粗利率 | - | 月次 |
| 営業利益率 | - | 四半期 |

## 連携部門

- **営業部**: 売上データ
- **管理部**: 経費管理
- **CEO**: 経営判断のための数値提供
