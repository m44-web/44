# 意思決定ログ

会社にとって重要な決定を記録します。あとから「なぜこうしたか」を振り返れるようにするためのものです。

## ファイル命名規則

`NNNN_タイトル.md`（連番）

例: `0001_社名をAXEに決定.md`

## 記録フォーマット

```markdown
# 決定: タイトル

- **日付**: YYYY-MM-DD
- **決定者**: オーナー / 心
- **ステータス**: 承認済み / 検討中 / 却下

## 背景
なぜこの決定が必要だったか

## 選択肢
1. 案A - メリット / デメリット
2. 案B - メリット / デメリット

## 決定内容
何をどうするか

## 理由
なぜこの選択肢を選んだか
```
