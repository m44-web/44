# 財務ルール：API費用の予算管理

**作成日**: 2026-04-17
**ステータス**: 承認済み

---

## 基本方針

AXEは資本金ゼロでスタートした。**1円も無駄にしない**を基本方針とする。

特にClaude API / Whisper APIは従量課金で、監視を怠ると一気に赤字になる。
以下の予算ルールを厳守する。

---

## 月次予算（2026年 Q2）

| 項目 | 月額上限 | 現状 |
|------|---------|------|
| Vercel | 0円（Hobby無料枠） | 無料枠内 |
| Supabase | 0円（Free Tier） | 未使用 |
| ドメイン | 100円/月（年額1,200円を月割り） | 契約済み |
| Claude API | 30,000円 | 未使用 |
| Whisper API | 5,000円 | 未使用 |
| その他 | 0円 | — |
| **合計上限** | **35,100円/月** | |

---

## アラートと自動停止

### レベル1: 注意喚起（予算50%到達）

- 月中にClaude API利用額が15,000円を超えたら心がオーナーにSlack通知
- 利用ペースを再確認し、残り日数でどう使うか計画

### レベル2: 警告（予算80%到達）

- 24,000円超で警告レベル
- 新規無料トライアルの受付を一時停止
- 既存ユーザーのみサービス継続

### レベル3: 強制停止（予算100%到達）

- 30,000円到達でAPI呼び出しを自動停止
- 翌月1日まで再開しない（請求月を跨ぐ）

## 1タスクあたりの原価目安

| タスク種別 | モデル | 平均原価 |
|-----------|-------|---------|
| FAQ応答（短文） | Claude Haiku | 1〜3円 |
| FAQ応答（資料検索付き） | Claude Sonnet | 5〜10円 |
| 議事録作成（30分会議） | Whisper + Claude | 30〜80円 |
| 日報集計（10人チーム） | Claude Sonnet | 15〜30円 |

**赤字防止ルール**: 1タスクあたり原価が100円を超える処理は、正社員プラン（29,800円/月）以上のユーザーのみに提供。

---

## 売上発生後のルール変更

売上が月額10万円を超えた時点で、本ルールを以下のように更新予定：
- API予算上限を売上の30%までに引き上げ
- サポート用の有料ツール（Intercom等）検討可
- 月次でオーナーと予実レビュー
