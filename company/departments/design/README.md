# デザイン部

## ミッション

ユーザーに愛されるプロダクト体験とブランドの視覚的アイデンティティを創る。

## 責任範囲

- UI/UXデザイン
- ブランドデザイン・ガイドライン管理
- プロトタイピング
- デザインシステムの構築・維持
- マーケティングクリエイティブ制作

## KPI

| 指標 | 目標 | 頻度 |
|------|------|------|
| ユーザビリティスコア | - | 四半期 |
| デザインタスク完了率 | - | 週次 |
| ブランド一貫性スコア | - | 四半期 |

## 連携部門

- **開発部**: UI実装、デザインシステム
- **マーケティング部**: クリエイティブ制作
- **広報部**: ブランド管理
