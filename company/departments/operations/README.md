# 事業運営部

## ミッション

SaaSプロダクトの安定運用とサービス品質の維持・向上を実現する。

## 責任範囲

- サービスの運用・監視
- カスタマーサポート
- オンボーディングプロセスの管理
- SLA（サービスレベル契約）の管理
- インシデント対応
- 運用プロセスの改善

## KPI

| 指標 | 目標 | 頻度 |
|------|------|------|
| サービス稼働率 | 99.9% | 月次 |
| 顧客満足度（CSAT） | - | 月次 |
| サポート応答時間 | - | 週次 |
| オンボーディング完了率 | - | 月次 |
| インシデント解決時間 | - | 月次 |

## 連携部門

- **開発部**: バグ報告、システム改善要望
- **営業部**: 顧客フィードバック共有
- **管理部**: 業務プロセス連携
