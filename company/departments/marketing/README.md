# マーケティング部

## ミッション

## 責任範囲

- ブランド戦略・ポジショニング
- コンテンツマーケティング（ブログ、SNS、ホワイトペーパー）
- SEO/SEM施策
- リードジェネレーション
- Webサイトの最適化
- イベント・ウェビナー企画
