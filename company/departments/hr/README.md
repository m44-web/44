# 人事部

## ミッション

最適な人材・AIリソースの確保と組織の成長を支える。

## 責任範囲

- 組織設計・組織図の管理
- 採用計画の策定・実行
- 役割定義・JD（ジョブディスクリプション）の管理
- 人材育成・研修
- 労務管理
- 組織文化の醸成

## KPI

| 指標 | 目標 | 頻度 |
|------|------|------|
| 採用充足率 | - | 四半期 |
| オンボーディング完了率 | - | 月次 |
| 組織カバレッジ率 | - | 四半期 |

## 連携部門

- **全部門**: 人材ニーズのヒアリング
- **管理部**: 労務管理
- **財務部**: 人件費管理
