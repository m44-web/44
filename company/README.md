# 合同会社LIT - AIエージェント/SaaS事業

## ミッション

AIエージェント技術を活用し、企業の業務を根本から変革するプロダクトを創る。

## ビジョン

AIエージェントが当たり前に働く世界をつくり、人がより創造的な仕事に集中できる社会を実現する。

## 事業概要

AIエージェントを核としたSaaSプロダクトの企画・開発・運営を行う。

### コアバリュー

1. **自律性** - AIも人も、自ら考え動く組織
2. **データドリブン** - 数字で語り、数字で判断する
3. **スピード** - 小さく速く回し、学びを最大化する
4. **透明性** - 全ての判断には理由があり、記録される
