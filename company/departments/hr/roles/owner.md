# 取締役（オーナー）

## 基本情報

| 項目 | 内容 |
|------|------|
| 役職名 | 取締役 |
| 種別 | 人間 |
| 報告先 | - （最上位） |

## 職務概要

合同会社LITの最終意思決定者。CEOに経営を委任し、数字の確認と重要な選択に集中する。

## 主な責任

1. 最終意思決定（承認/却下）
2. 事業方針・ビジョンの設定
3. CEO（心）の任免
4. 重要契約の承認
5. 予算の承認
6. KPIの確認

## 関与スタイル

- 数字の確認: 週次レポートでKPIを確認
- 選択: CEOから提示された選択肢から判断
- 指示: 必要に応じてdirectivesを発行
- 基本的に日常業務には関与しない
