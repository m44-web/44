# SaaSビジネス ベストプラクティス

## 重要指標

### 成長指標

| 指標 | 定義 | 健全な水準 |
|------|------|-----------|
| MRR | 月次定期収益 | 前月比10-20%成長（初期） |
| ARR | 年間定期収益 | MRR x 12 |
| 顧客数 | アクティブ顧客数 | 純増が続くこと |
| ARPU | 顧客あたり平均収益 | 上昇傾向 |

### 効率指標

| 指標 | 定義 | 健全な水準 |
|------|------|-----------|
| CAC | 顧客獲得コスト | LTVの1/3以下 |
| LTV | 顧客生涯価値 | CACの3倍以上 |
| Churn Rate | 月次解約率 | 3%以下 |
| NRR | 売上維持率 | 100%以上 |

### 財務指標

| 指標 | 定義 | 健全な水準 |
|------|------|-----------|
| Burn Rate | 月次資金消費額 | ランウェイ12ヶ月以上 |
| Gross Margin | 粗利率 | 70%以上 |
| Rule of 40 | 成長率+利益率 | 40%以上 |

## プライシング戦略

1. **バリューベースプライシング**: 顧客が得る価値に基づく
2. **ティア構成**: 3段階が基本（Starter / Pro / Enterprise）
3. **フリーミアム vs トライアル**: ターゲットにより選択
4. **年間契約割引**: 10-20%のディスカウント

## グロース戦略

1. **PLG（Product-Led Growth）**: プロダクト体験で顧客を獲得
2. **コンテンツマーケティング**: 専門知識でリードを獲得
3. **パートナーシップ**: 補完的なサービスとの連携
4. **カスタマーサクセス**: 既存顧客の拡大（Expansion Revenue）
